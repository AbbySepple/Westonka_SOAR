Make sure to follow these steps to replace old class and homeroom lists with new class and homeroom lists.

  1. Drop all current tables

    2. Only drop current tables if this is NOT the first time you are using this application. The first time this application is used, the tables will not exist.



  2. Upload the homeroom data

    3. In this CSV file, there should be the following columns with these headers:

    identifier, grade

    Where identifier is the homeroom class number and the grade is one of five choices:

    1. KD
    2. 01
    3. 02
    4. 03
    5. 04

         Below are examples of how certain rows should look in the CSV file:

    1. 123, KD
    2. 26, 01
    3. 106, 02
    4. 14, 03
    5. 111, 04





  3. Upload User Data:

    4. Upload the CSV file that has all of the user data, including students, teachers, and admins

    In this CSV file, there should be the following columns with these headers:

    name, email, admin, teacher, homeroom_id



    Below are examples of how certain rows should look in the CSV file:


    1. Student

      2. Full Name, student@277apps.com, false, false, 1

    2. Teacher

      3. Full Name, teacher@277apps.com, false, true, 1

    3. Admin

      4. Full Name, admin@277apps.com, true, false,
